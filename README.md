# tfonics

A collection of functions and utilities for the tfonics project.

To add this project to your poetry project, run

```bash
poetry add --git git@git.ecdf.ed.ac.uk:oalmelid/tfonics.git
```

# Code standards

Linting with pylint is enforced on all merges to master.

## Unit tests

Unit tests can be found in [the test folder](https://git.ecdf.ed.ac.uk/oalmelid/tfonics/tree/master/tests), we use the pytest framework.
